### Commands

Command | Description
------- | -----------
`yarn build` | Generate files in the `dist` folder
`yarn release` | Start the process to release a new version

### Workflow

1. Make changes
2. `yarn build`
3. `cd example`
4. `yarn refresh`
5. `yarn test`
6. Verify that everything works
7. Push `master` or make `PR`
8. `yarn publish`
